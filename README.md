# Pet Clinic Infrastructure

This repository contains the infrastructure as code (IaC) for deploying and managing the Pet Clinic application.

## Overview

The Pet Clinic application is a sample Spring Boot application that demonstrates various features and best practices. This repository contains the infrastructure code needed to deploy and run the application in AWS.

## Architecture

The infrastructure includes:

- Amazon EKS for container orchestration
- Amazon RDS for the database
- Amazon VPC for networking
- Amazon CloudWatch for monitoring and logging
- AWS IAM for security and access control
